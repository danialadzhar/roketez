# Roketez

E-Learning Momentum Internet

# Installation 

1. Clone Project

```
https://github.com/danialadzhar/roketez.git
```

2. Dalam Folder Momentumlearning, Run :

```
composer install
php artisan key:generate
php artisan migrate
```

3. Tukar .env.example kepada .env:

```
cp .env.example .env
```

4. Tukar details database dalam .env
5. Run project :

```
php artisan serve
```
